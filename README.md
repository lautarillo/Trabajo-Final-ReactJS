# Lautarillo Reacciona

SPA de e-commerce hecha con React y Vite.

## Incluye

- Navegacion con React Router: catalogo, categorias, detalle, carrito y checkout.
- Componentes contenedores y de presentacion: ItemListContainer, ItemList, ItemDetailContainer, ItemDetail.
- ItemCount con minimo, maximo por stock y botones deshabilitados en estados invalidos.
- Ocultamiento de ItemCount luego de agregar un producto al carrito.
- Carrito global con React Context.
- CartWidget con icono y total de unidades.
- Cart con productos, cantidades, subtotales y total final.
- Checkout con datos del comprador y orden local simulada con ID visible.
- Loaders y estados condicionales para carga, carrito vacio, producto sin stock y producto no encontrado.

## Ejecutar

```bash
npm install
npm run dev
```

Luego abrir la URL local que indique Vite.
